# LSAP Website

A minimalist, geek-style website for the Language Server Agent Protocol.

## Design Philosophy

- **Minimalist**: Clean lines, comfortable spacing, low-saturation colors
- **Geek Style**: Monospace fonts, technical aesthetics, developer-focused
- **Human-Centric**: Low-saturation blue-gray palette for visual comfort
- **Typography**: IBM Plex Mono + Crimson Pro

## Tech Stack

- **Framework**: React 19 + Vite
- **Styling**: Tailwind CSS
- **UI Components**: shadcn/ui
- **Routing**: React Router
- **Icons**: Lucide React
- **Markdown**: React Markdown + remark-gfm

## Development

```bash
# Install dependencies
bun install

# Start dev server
bun run dev

# Build for production
bun run build

# Preview production build
bun run preview
```

## Features

- Interactive examples showing LSAP workflow
- Animated step-by-step demonstrations
- Documentation viewer with markdown rendering
- Responsive design
- Smooth transitions and micro-interactions
